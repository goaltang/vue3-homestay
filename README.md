# 民宿预订系统

## 项目介绍

基于 Spring Boot + Vue3 的民宿预订系统，提供房源展示、预订管理等功能。

## 技术栈

- 后端：Spring Boot, Spring Security, JPA
- 前端：Vue3, Element Plus
- 数据库：MySQL

## 本地开发

1. 克隆项目

   ```bash
   git clone https://github.com/你的用户名/homestay-system.git
   ```

2. 运行后端

   ```bash
   cd backend
   mvn spring-boot:run
   ```

3. 运行前端
   ```bash
   cd frontend
   npm install
   npm run dev
   ```
